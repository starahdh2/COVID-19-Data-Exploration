# COVID-19 Data Exploration 🦠📊

## Project Overview

This project analyzes COVID-19 data of Pakistan using Python.  
The dataset is taken from Our World in Data and analyzed using Pandas and Matplotlib.

---

## Tools & Technologies

- Python
- Pandas
- Matplotlib
- VS Code

---

## Dataset

Dataset Source:
https://catalog.ourworldindata.org/garden/covid/latest/compact/compact.csv

File Used:

- compact.csv

---

## Tasks Performed

- Loaded COVID-19 dataset
- Filtered Pakistan data
- Converted date column into datetime format
- Plotted COVID-19 total cases over time
- Calculated death rate
- Generated insights from the dataset

## Graph

The project generates a line graph showing COVID-19 total cases in Pakistan over time.

---

## Death Rate Formula

Death Rate = (Total Deaths / Total Cases) × 100

---

## Insights

1. COVID-19 cases increased rapidly during pandemic waves.
2. Death rate remained lower compared to total cases.
3. Cases reduced after vaccination campaigns and safety measures.
